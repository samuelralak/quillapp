# Quill

**Conversational forms (Typeform-style), powered by Nostr, private by default.**

A single self-contained HTML prototype. Makers build a form, publish it as a Nostr event, and read responses that are sealed so only their key can open them. Respondents answer one calm question at a time and need **no account, no signer, and no key**: under the hood each submission is gift-wrapped to the form owner with a throwaway ephemeral key.
